# Plant Disease Recognition - AI Model Project

## Project Overview

Plant Disease Recognition is an innovative AI-powered mobile application that helps farmers and gardeners instantly identify plant diseases using deep learning technology. The project leverages TensorFlow and computer vision to analyze plant leaf images and provide accurate disease detection with bilingual support (English and Hindi).

## Key Features

### 🤖 AI-Powered Disease Detection
- **Deep Learning Model**: Built with TensorFlow/Keras using convolutional neural networks
- **High Accuracy**: Trained on comprehensive plant disease datasets
- **39 Disease Classes**: Covers major crops including apple, corn, grape, tomato, potato, and more
- **Instant Results**: Real-time disease prediction with confidence scores

### 📱 Mobile-First Design
- **Android App**: Native WebView application for easy deployment
- **Touch-Friendly Interface**: Optimized for mobile devices
- **Image Upload**: Support for camera capture and gallery selection
- **Offline Capabilities**: Local model processing for privacy

### 🌍 Bilingual Support
- **English & Hindi**: Complete disease information in both languages
- **Localized Content**: Symptoms, causes, and treatment details in Hindi
- **Cultural Adaptation**: Tailored for Indian agricultural context

### 📊 Comprehensive Disease Information
- **Detailed Symptoms**: Visual and text descriptions
- **Cause Analysis**: Understanding disease origins
- **Treatment Options**: Chemical and organic solutions
- **Prevention Methods**: Proactive disease management
- **Severity Assessment**: Help prioritize treatment urgency

## Technology Stack

### Backend
- **Flask**: Python web framework for API development
- **TensorFlow**: Deep learning framework for model deployment
- **NumPy**: Numerical computing for image processing
- **Flask-CORS**: Cross-origin resource sharing for mobile app

### Frontend
- **Android WebView**: Native mobile application wrapper
- **Bootstrap**: Responsive web design framework
- **HTML5/CSS3/JavaScript**: Modern web technologies

### Model Architecture
- **Input**: 160x160x3 RGB images
- **Network**: Convolutional Neural Network (CNN)
- **Output**: 39-class disease classification
- **Performance**: Optimized for mobile deployment
